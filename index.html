The free resource for learning about HaxeFlixel.

This website is a collection of snippets that demonstrate some of the features of HaxeFlixel.

HaxeFlixel is a free, open-source, cross-platform, 2D game development library that uses OpenFL and Haxe, and is based on the original AS3 Flixel by Adam "Atomic" Saltsman.

This website aims to assist anyone who wants to use HaxeFlixel in understanding and utilizing the many powerful concepts this library has to offer.
